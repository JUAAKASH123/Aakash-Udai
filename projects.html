VirtuFit is a real-time AI-powered virtual dressing room that lets users try on clothes digitally via a live webcam feed. The system uses computer vision and pose estimation to detect body landmarks and dynamically overlay clothing aligned to the user's body. By combining pose tracking, gesture interaction, and image processing, it creates an interactive augmented reality-style try-on experience.

Key Features
  • Real-Time Virtual Try-On — Clothing dynamically overlaid via live webcam
  • AI Pose Detection — MediaPipe Pose tracks shoulder and hip positions
  • Dynamic Cloth Alignment — Size and position from shoulder width & torso length
  • Hand Gesture Controls — Switch outfits with MediaPipe Hands gestures
  • Frame Stabilization — Smoothing techniques reduce jitter across frames
  • Alpha Blending — Clothes seamlessly blended using OpenCV

  • Maintaining low latency for real-time interaction

The AI Teacher Lesson Plan Creator is a Retrieval-Augmented Generation (RAG) system that automatically generates structured lesson plans from educational PDFs and textbooks. It retrieves relevant content via hybrid search (keyword + semantic) and generates context-aware lesson plans using Gemini LLMs. Teachers enter the subject, class, and topic — the system produces plans with objectives, activities, explanations, and assessments.

Key Features
  • Automated Lesson Generation — Full plans with objectives, activities & assessments
  • RAG Pipeline — Document retrieval combined with generative AI
  • Hybrid Search — Keyword + semantic vector search for accurate retrieval
  • Cross-Encoder Re-Ranking — Better contextual accuracy before generation
  • Document Knowledge Base — Upload PDFs converted to searchable embeddings
  • FastAPI Backend — Scalable APIs for document processing and generation

Twitter Sentiment Analysis

Multi-class sentiment analysis on Twitter data using DistilBERT fine-tuned with LoRA (Low-Rank Adaptation) for parameter-efficient training. The system classifies tweets into positive, negative, and neutral sentiment — achieving 87% F1 score while keeping compute and memory requirements minimal.

Key Features
  • DistilBERT Transformer — Lightweight model for accurate classification
  • LoRA Fine-Tuning — Reduces training cost via low-rank matrix injection
  • Multi-Class Sentiment — Positive, negative, and neutral detection
  • 87% F1 Score — Strong performance on real-world Twitter data
  • Efficient Training — Minimal compute overhead vs full fine-tuning
Model Pipeline
1
Data CollectionLabeled Twitter sentiment dataset
2
Text PreprocessingTokenization, cleaning, normalization
3
EmbeddingDistilBERT generates contextual embeddings
4
LoRA Fine-TuningParameter-efficient adaptation applied
5
ClassificationModel predicts sentiment category
Why LoRA?
  • Injects low-rank matrices into transformer layers — far fewer trainable params
  • Achieves comparable accuracy to full fine-tuning at a fraction of compute cost
  • Ideal for resource-constrained environments without dedicated GPU clusters
  • Enables fast experimentation and iteration cycles
